# Telco Customer Churn Analysis

## Project Overview

This project analyzes customer churn data from a telecommunications company using Python. The goal is to understand why customers leave, identify patterns in customer behavior, and provide insights that can help reduce churn.

## Features

- Clean and prepare the dataset
- Calculate the overall customer churn rate
- Analyze customer tenure
- Group customers into tenure cohorts
- Compare churn across different customer groups
- Visualize trends using charts
- Provide business insights and recommendations

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- Matplotlib

## Installation

Install the required libraries before running the notebook.

```bash
pip install pandas matplotlib notebook
```

## How to Run

1. Download or clone this repository.

```bash
git clone <repository-link>
```

2. Move into the project folder.

```bash
cd <project-folder>
```

3. Start Jupyter Notebook.

```bash
jupyter notebook
```

4. Open the notebook:

```
Telco_Churn_Analysis.ipynb
```

5. Run all the cells from top to bottom.

## Dataset

This project uses the **Telco Customer Churn** dataset, which contains customer information such as demographics, services subscribed to, tenure, monthly charges, total charges, and whether a customer left the company.

**Source:**  
https://www.kaggle.com/datasets/blastchar/telco-customer-churn
